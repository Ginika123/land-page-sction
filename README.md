 🪑 Panto Furniture Landing Page

📖 Project Overview

This is a simple **furniture landing page website** built using **HTML, CSS, and JavaScript**.
The project was created as part of my learning journey in **frontend web development**.

The website includes a navigation bar with a dropdown menu, a hero section with a background image, and an information section with interactive buttons.

---
 ✨ Features

* Navigation bar with dropdown menu
* Hero section with background image
* Interactive **"More Info"** buttons
* Clean layout using **CSS Flexbox**
* Basic JavaScript interactivity

---

 🛠 Technologies Used

* **HTML5** – for structuring the webpage
* **CSS3** – for styling and layout
* **JavaScript** – for dropdown menu and button interactions
